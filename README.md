# Rock Paper Scissors Game

A simple and interactive **Rock Paper Scissors** web game built using **HTML**, **CSS**, and **JavaScript**. This project allows users to play against the computer with instant results and a smooth user experience.

**Live Demo:** https://saisurya01.github.io/Rock-Paper-Scissors/

---

## Overview

This Rock Paper Scissors game is a browser-based implementation of the classic game. The user selects an option, the computer randomly chooses its move, and the result is displayed immediately.

The project demonstrates core JavaScript concepts such as event handling, conditional logic, and DOM manipulation.

---

## Features

- Play Rock, Paper, or Scissors against the computer
- Instant result display (Win / Lose / Draw)
- Random computer move generation
- Simple and clean user interface
- Fully responsive design

---

## Technologies Used

- HTML5
- CSS3
- JavaScript (ES6+)
